------Project Name:
Food Delivery Database System 🍔

-----What it does:
This project manages a food delivery system using SQL. It stores data of customers, restaurants, menu items, orders, delivery agents, payments, and reviews. It also helps to track orders, calculate revenue, and analyze data like most ordered items and top restaurants.

-----How to run:

Open MySQL / any SQL tool
Run the SQL file step by step (create database → tables → insert data → queries)

-----Output:

Shows customer orders with restaurant details
Finds top restaurants and revenue
Displays delivery status
Shows most ordered items
Lists pending orders and customer activity
